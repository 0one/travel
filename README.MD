# 自驾8条金规
- 疲劳驾驶是大忌
- 选择车型按路线
- 车况检查要仔细
- 证件带齐不能忘
- 自驾设备要齐全
- 精力充沛是保障
- 出游不忘做环保
- 开心放松是目的
